prototype for my puzzle platformer in Godot.

## instructions:
- Go to releases page
- Download the ZIP file that corresponds to your OS and CPU
- extract the files from the zip to somewhere you'd like. *MAKE SURE ALL FILES IN THE ZIP ARE EXTRACTED TO THE SAME DIRECTORY OR THE GAME WONT LOAD!*
- open the executable file (.exe on Windows, .x86_64 on linux)
- enjoy! report bugs and send feedback on gameplay, **preferably record footage of your blind run**
